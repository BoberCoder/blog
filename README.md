# blog

URL's :

POST - /post/add (add record)

GET - /post (show all records), 

/post/{id} (show record on ID)

DELETE - /post/{id} (delete record on ID)

PUT - /post/{id} - (edit record on ID)

PATCH - post/{id} - (patch record on ID)
